Worm

Worms are self replicating malware that can make copies of itself without the use of human interaction and do not need to attatch themselves to software in order to cause damage.

A worm is a third form of malware, which only needs user interaction to gain access and launch an application in order to worm its way in, so to speak. Once it has this access, it no longer needs user input to attain its programmed goals.

A worm seeks out vulnerabilities in a computer's operating system that allow it to install and make copies of itself. It will then follow the known vulnerabilities in the network and file transfer protocols.

An obvious way to keep safe from worms is updating operating systems. Sound sinple? Well, you might be surprised to know that there are a number of enterprises and individual's that don't do this, giving worms a cyber haven of holes to burrow through.
